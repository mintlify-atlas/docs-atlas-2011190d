<Warning>
The Doble Corrediza system is not yet implemented. Selecting it in the application displays a "coming soon" screen. No configuration or calculation is available at this time.
</Warning>

Doble Corrediza is a planned window system where both horizontal panels slide independently, leaving no fixed central panel. This layout maximises the usable opening area and is suited to situations where full ventilation or wide access is a priority — such as balcony-adjacent rooms, service counters, and pass-through openings.

## Difference from Sistema Nova

Sistema Nova uses a **mixed** layout: fixed panels frame the opening and one or more sliding panels travel within that frame. Doble Corrediza eliminates the fixed panels entirely — both leaves slide, overlapping in the centre when closed and parting to either side when open.

| Feature | Sistema Nova | Doble Corrediza |
|---|---|---|
| Fixed panels | Yes | No |
| Sliding panels | 1–3 | 2 (both leaves) |
| Maximum clear opening | Partial | Full width |
| Implementation status | Available | Coming soon |

## Use in the meantime

For sliding panel requirements, use **Sistema Nova** with the minimum number of fixed panels. A 1-corrediza + 2-fijos configuration provides the closest available equivalent.

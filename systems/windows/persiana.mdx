The Persiana system is not yet implemented. Selecting it in the application displays a "coming soon" screen. No configuration or calculation is available at this time.
</Warning>

Persiana is a planned window system for MediPro that will handle enrollable (roller) blind configurations. The system targets aluminum-framed roller blinds where a fabric or slat panel winds onto a concealed roller housed in a top cassette.

## Planned behaviour

When implemented, the Persiana configurator will accept the window opening dimensions and calculate:

- **Cassette** housing profile length and mounting hardware
- **Roller tube** diameter and length relative to opening width
- **Bottom rail** profile for tensioning the blind
- **Side guide** profiles for lateral containment
- **Fabric or slat** cut width and drop length with hem allowances

The system is intended for both interior privacy blinds and exterior solar-control screens, covering residential and light commercial applications.

## Use in the meantime

The currently available window system is **Sistema Nova**, which provides a full sliding + fixed panel configuration with automatic profile calculations.
